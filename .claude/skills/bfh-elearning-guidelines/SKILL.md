# BFH E-Learning Förderprogramm Richtlinien

## Übersicht

Das E-Learning-Förderprogramm der BFH unterstützt die Entwicklung aufwändigerer E-Learning-Projekte. Es fördert didaktisch sinnvolle und nachhaltige Projekte mit Vorbildcharakter. Das Programm finanziert Arbeitsstunden, keine technische Ausstattung.

## Kurzübersicht

| Parameter | Wert |
|-----------|------|
| Förderumfang/Jahr | CHF 100'000 (Stundengutschriften) |
| Typische Fördersumme | CHF 10'000 (höher in begründeten Fällen) |
| Mindest-ECTS | 2 ECTS |
| Pflicht-Tools | Moodle, Kaltura |
| Einreichfrist | 30. April (Call 2025), 31. Juli (Folgejahre) |
| Abschlussprodukt | OER + Präsentation |

## Stundenansätze

| Rolle | Ansatz (CHF/h) |
|-------|----------------|
| Dozierende | 104 |
| Wissenschaftliche Mitarbeitende | 70 |
| Assistierende | 52 |

## Bewertungskriterien

| Kriterium | Gewichtung | Kernfrage |
|-----------|------------|-----------|
| Strategie-Bezug | Hoch | Bezug zu Vielfalt, Future Skills oder Vernetzung? |
| Innovation | Hoch | Welche didaktischen Neuerungen werden eingeführt? |
| Qualitätsverbesserung | Hoch | Wie verbessern digitale Elemente die Lehre? |
| Nachhaltigkeit | Mittel | Wie wird langfristige Nutzung sichergestellt? |
| Vernetzung | Mittel | Wie fördert das Projekt Zusammenarbeit? |
| Transferpotenzial | Mittel | Kann das Konzept übertragen werden? OER möglich? |

## Schreibstil-Richtlinien

### DO
1. **Konkret sein**: Vermeide vage Aussagen; nutze konkrete Beispiele und Metriken
2. **Didaktischen Mehrwert betonen**: Zeige klar, wie die Innovation das Lernen verbessert
3. **Nachhaltigkeit nachweisen**: Erkläre, wie das Projekt über die Förderperiode hinaus Bestand hat
4. **Vernetzung aufzeigen**: Beschreibe Kooperationen mit anderen Dozierenden/Departementen
5. **Transferpotenzial hervorheben**: Erkläre, wie andere von den Materialien profitieren können

### DON'T
1. Technische Ausstattung beantragen (nicht förderfähig)
2. Projekt ohne Einbettung in konkrete Lehrveranstaltung vorschlagen
3. Stundenansätze anders als BFH-Vorgaben verwenden
4. Förderungswürdigkeit ohne Bezug zu allen 6 Kriterien begründen
5. OER-Bereitschaft verschweigen

## Sprache

Anträge müssen auf **Deutsch** verfasst werden.
Verwende **Schweizer Rechtschreibung** (kein ß, verwende ss).

## Voraussetzungen für Förderung

- [ ] Lehrveranstaltung mit mindestens 2 ECTS
- [ ] Nutzung von BFH-Tools (Moodle, Kaltura)
- [ ] Kompetenz-, handlungs- und praxisorientiertes Konzept
- [ ] Bereitschaft zur OER-Publikation
- [ ] Bereitschaft zur Präsentation als Good-Practice
